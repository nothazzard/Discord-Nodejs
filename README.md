# Discord BOT - Nodejs

O Discord Bot é um bot modular desenvolvido em Node.js, focado em simplicidade e eficiência.
Possui comandos básicos, arquitetura escalável e integração rápida com servidores Discord por meio da API oficial.
Perfeito como ponto de partida para desenvolvedores ou como utilitário leve para servidores menores.

Para instalação do BOT simples basta fazer a instalação por :

npm install
npm start

Pronto agora o seu bot está rodando, lembre-se de configurar **.env**.
